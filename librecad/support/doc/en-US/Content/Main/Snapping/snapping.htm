Snapping

Use different snap/restrict tools to control some precision aspects of your drawing, such as snapping on endpoints or entities, to achieve contiguity or tangency. Snap tools are available in the bar in the bottom left area of the application.

Tool Icon Command Description
Exclusive Snap Mode none When enabled, only one snap mode is allowed to be ON at a time.
Free Snap os, sf Allows for the crosshair to move freely while other snap modes are enabled.
Snap on Grid sg Snap to a grid intersection.
Snap on Endpoints se Snap to the endpoints of a line segment, the quadrants of a circle, a point, or the alignment point of a text or mtext object.
Snap on Entity np, sn Snap to the path of an entity.
Snap Center sc Snap to the center of a circle or ellipse. It will also snap to the foci of an ellipse.
Snap Middle sm

Snap to the middle of a path.

If the Middle Points value is 2, you can snap to the trisection points of a line segment.

 

Tool options (in the toolbar above the drawing area):
Snap Distance sd

Snap to the endpoint of a line segment from a specified distance.

Example: If you snap to the endpoint of a line segment and then activate Snap Distance with a Distance input of 50, then it will snap to a point 50 units from the endpoint on the line segment. However, it will also snap to a point that is 50 units from the other endpoint.

Tool options (in the toolbar above the drawing area):
Snap Intersection si Snap to the intersection of two entities. Note this does not currently work for polylines.
Restrict Horizontal rh Restricts the placement crosshairs to the x-axis (horizontal movement).
Restrict Vertical rv Restricts the placement crosshairs to the y-axis (vertical movement).
Restrict Orthogonal rr Restricts the placement crosshairs to the x or y-axis. (horizontal or vertical movement).
Restrict Nothing   rn

Turns off all restricted cursor movements (horizontal, vertical, and orthogonal).

This command is only available via Command line.
Set Relative Zero Position rz, setrelativezero Manually sets the Relative Zero Point at the selected coordinate.
Lock Relative Zero Position none Locks the Relative Zero Point to the current coordinate.
