Using Layers

Creating layers

  1. Click Add Layer ( ) on the Layer List dock.
  2. Enter a name for the layer. Additionally, indicate if the layer is a Construction Layer, and adjust the Default Pen options as needed.
  3. Click OK.

Removing layers

Removing a layer will remove the layer and all geometry that exists on that layer.

  1. Click the layer you want to remove on the Layer List dock.
  2. Click Remove Layer (  ), or right-click and click Remove Layer.
  3. Click OK.

Making a layer active

  • Simply click a layer on the Layer List dock to make it active.

Modifying the color of a layer and other attributes

  1. Click the layer you want to modify.
  2. Click Modify Layer (  ).
  3. Make adjustments to any of the layer settings.
  4. Click OK.

Changing the layer of an entity

  1. Click one or more entities.
  2. Click either Attributes ( ) or Properties ( ) from the Modify tools.
  3. Click a different layer from the Layer dropdown menu.
  4. Click OK.

Ordering layers

Layers are displayed in alpha-numeric order in the layer list. However, the order of the layers do not relate to the order that entities appear in the drawing. Use the commands in Modify Tools > Order button ( ) to move entities up or down.

Action Key Result
Move to top [Home] Moves the selected entity to the top most position.
Move to bottom [End] Moves the selected entity to the bottom most position.
Raise over entity [Page Up] Moves the selected entity up one relative position.
Lower over entity [Page Down] Moves the selected entity down one relative position.

Creating construction layers

To set an existing layer as a construction layer:

  • Click Construction Layer () beside the layer name.

When this button is green, it means that the layer is set as a construction layer. When the button is grayed out, the layer is not a construction layer.

Create new layers that do not have an assigned process in ProNest to use as construction layers. This will avoid importing into ProNest any unwanted geometry from construction layers.

For example, if a cut layer is used as a construction layer to create construction lines, the line entities used to create the construction lines will be recognized as geometry in ProNest. However, if a new layer is used that ProNest doesn't recognize, lines used to create construction lines will be ignored by ProNest.

Creating construction lines

To create construction lines of infinite length:

  1. Ensure your construction layer is active (  ).
  2. Create a line using any of the Line or Polyline Tools. A new line of infinite length will be drawn as a preview over this line.

    3. If you are placing construction lines on a layer with an assigned process in ProNest, you'll need to delete line entities used to create construction lines before importing the drawing into ProNest. It is recommended that you create new layers without an assigned process in ProNest to use as construction layers to avoid importing into ProNest any unwanted geometry from construction layers.

  3. Adjust your line, then click anywhere in the drawing to place the line. This selection will become the reference point to create another construction line.

  4. Place as many construction lines as needed, repeating step 3.

  5. When you are done drawing construction lines, right-click or select the Esc key to exit the construction line creation mode.

Hiding construction lines

To make construction lines invisible while keeping all other geometry on a layer visible:

  • Click Construction Layer () beside the layer name.

The construction layer button will gray-out, hiding all construction lines.

Any construction lines you have created will not be removed by toggling the construction layer button, they will only be hidden.

Removing construction lines

  1. Click one or more construction lines.
  2. Select the Delete key.

Following these steps will delete any infinite construction lines but not any geometry that was used to create construction lines (lines and polylines).
