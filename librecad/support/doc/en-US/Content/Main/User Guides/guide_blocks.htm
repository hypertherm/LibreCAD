Using Blocks

Blocks are small, reusable drawings of commonly used objects such as bolts, title blocks, and so forth. Inserted blocks, also called inserts, are composed of geometric shapes (lines, arcs) and can also include text and dimensions. LibreCAD includes several libraries of blocks that can be inserted into a drawing or you can create your own.

The are two dock widgets for managing blocks:

  • The Block List dock for managing blocks within the current drawing, and
  • Library Browser dock that shows the collection of blocks available and allows users to insert blocks into the current drawing.

Read more about Dock Widgets here and the Library Browser here.

When inserted into a drawing, a block is treated as a single entity. Blocks can be inserted from the Block List into the same drawing more than once with different locations, scales, and rotation angles. Inserted blocks are all linked. Changes to one inserted block will be reflected in all instances of the block.

Creating a Block from an existing drawing

  1. Draw the object that you want to convert into a block.
  2. Select all entities that will make up the block. Do so by clicking individual entities or selecting multiple entities in an area.
  3. Click Create Block ( ) from the Block List dock.
  4. Specify a reference point.
  5. Name the block, and click OK.

The block now appears in the Block List and can be inserted into the drawing multiple times or saved to a file or to the Library.

Creating a Block from an empty Block

  1. Click Add an empty block ( ) from the Block List dock.
  2. Name the block, and click OK.
  3. Click Edit the active block in a separate window ( ). This will open a blank block drawing document tab.
  4. Draw the block. The drawing origin (0,0) will be the reference point for the block.
  5. Exit the block drawing tab. The block will be saved and can be inserted into a drawing.

Inserting Blocks into a drawing

  1. Click the block you want to insert from the Block List.
  2. Click Insert the active block ( ) from the Block List dock.
  3. Specify the insertion point by clicking somewhere in the drawing, or by specifying coordinates in the Command Line.

Inserting Blocks into a drawing using the Toolbar (Advanced)

  1. Click the block you want to insert from the Block List.
  2. Click Insert the active block ( ) from the Block List dock.
  3. Specify parameters that appear on the toolbar.

  • Parameters related to block transformations: Angle of rotation and scale Factor.
  • Parameters related to block patterns: Array (grid size) and Spacing between columns and rows of the array.
  1. Specify the insertion point by clicking somewhere in the drawing, or by specifying coordinates in the Command Line.

Removing an instance of a block from a drawing

  1. Click the block on the drawing.
  2. Click the Delete key.

Removing a block and all instances from a drawing

Removing an instance of a block from a drawing will only remove the block that was placed on a layer, and the block will still be stored in the DXF file. To completely remove a block from both the drawing layer and the DXF file, do the following:

  1. Click the block on the Block List.
  2. Click Remove the active block ( ) on the Block List.

Exploding Blocks

You may want to explode blocks to modify some geometry or individual entities. When blocks are first inserted into a drawing, the block is treated as a whole and can't be modified unless it is exploded.

  1. Enter explode into the Command Line, or click Explode ( ) from the Modify options.
  2. Click the block you are exploding.
  3. Select the Enter key.

Modifying Blocks

If the block is saved to a file, simply open the file, make changes, and save before exiting.

If the block isn't saved to a specific drawing already and instead is part of the current drawing you're working in, you can modify the block by doing the following:

  1. Click the block you want to modify from the Block List.
  2. Click Edit the active block in a separate window (  ) from the Block List.
  3. Make any changes to the block.
  4. Exit the block drawing.

The block is updated.

Inserting Blocks from the Library

  1. From the Library Browser dock, click the block that you want to insert.
  2. Click Insert.
  3. Click a reference point in the drawing, or enter coordinates in the Command Line.

The block may be placed as many times as needed. Select the Esc key, or right-click to exit the block insertion mode.

Saving Blocks to a file

  1. Click the block you want to save from the Block List.
  2. Click Save the active block to a file ( ).
  3. Specify a name and save location for the file, then click OK.
