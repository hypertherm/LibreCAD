Using the Library Browser

Additional part libraries can be added for blocks or drawings that you create. Any DXF drawing listed in your Library folders is inserted as a block, which can be exploded into individual geometry.

Specifying a new library folder

  1. Click Options.
  2. Click Application Preferences.
  3. Specify a folder path in the Part Libraries field.

Do not place blocks directly in the parent parts library directory. Blocks must be placed in a sub-folder of the parent libraries folder to appear in the Library browser.

  1. Click OK.
  2. Click Refresh from the Library Browser.

The folder you specified in Application Preferences should appear in the Library Browser, along with the standard installed folders and drawings.

Adding files to your new library folder

Once you have added a new folder to your Library Browser (see steps in the previous section), you can add DXF drawings to that folder:

  • Copy and Paste files from the File Explorer on your computer into a sub-folder within the folder you specified in Application Preferences.

Files can't be added to the Library Browser from LibreCAD and instead must be added via the File Explorer on your PC.

Inserting a block from the Library Browser

  1. Click the block you want to insert.
  2. Click Insert.
  3. Specify a reference point in the Command Line, or select a point on the drawing.

In this mode, you can insert as many copies of the block as needed. Select the Esc key, or right-click to exit the block insertion mode.

Removing a user-specified library folder

  1. Ensure the folder path is removed from the Part Libraries field in your Application Preferences.
  2. Click Rebuild from the Library Browser.

The folder should no longer appear in your Library Browser.
