Mouse and keyboard controls, zooming, and panning

Mouse controls

Select an entity

Click on an entity once to select it.

Select multiple entities

Click multiple entities to select them. You do not need to hold down any additional keys.

Select all entities in the drawing

Once an entity is selected, click that entity again to deselect it.

Zoom to the extents of the drawing

Double-click the scroll wheel to automatically zoom to the extents of the drawing.

Open a menu of recently used tools

Right-click in the drawing area to open a menu of the 15 most recently used tools.

Keyboard controls

Select all entities in the drawing

Select Ctrl+A to select all entities in the drawing.

Deselect all entities

To deselect all selected entities, do either of the following:

  • Select Esc.
  • Select Ctrl+K.

Toggle snap options on/off

Select the Spacebar to toggle your snap options on/off. If this isn't working, check your Application Preferences.

Enter commands in the Command Line

From the main drawing screen, you can begin typing, which will automatically populate in the Command Line, without having to click in the text field first.

Scroll vertically in the drawing

To scroll up or down in your drawing without relying on zooming, do either of the following:

  • Hold down the Ctrl key while using the scroll bar
  • Use the scroll bar.

Scroll horizontally in the drawing

To scroll to the left or right in your drawing without relying on zooming, do either of the following:

  • Hold down the Shift key while using the scroll bar
  • Use the scroll bar.

Exit out of a tool mode

Select Esc to exit out of any tool mode.

Open the LibreCAD Help

Select F1 to open context-sensitive help.

View all Command Line commands

From the main drawing area, select Tab to return a list of all available commands for the Command Line.

Toggle full screen mode

Select F11 to enter/exit full screen mode.

Zooming

You can zoom in or out of a drawing by doing either of the following:

  • Use the scroll wheel on your mouse.
  • Click the Zoom In ( ), Zoom Out ( ), or Auto Zoom ( ) buttons, which are located below the main menu buttons at the top of the screen.

Additional options for zooming can be configured in the Application Preferences.

Panning

You can pan, or move the drawing screen around, by doing either of the following:

  • Hold down the scroll wheel so that it clicks, and drag the mouse to move the screen.
  • Click the Zoom Panning button ( ), which is located below the main menu buttons at the top of the screen, then click and drag the mouse to move the screen.
