Line tools

If you are seeing lines of infinite length being created when you insert lines, it is likely that the layer on which you are drawing is a Construction Layer.

Tool Icon Command Description

2 Points

 l, li, line

Draw a line between two assigned points.

Tool options:
Angle none

Draw a line from an assigned point defining the start, middle or end of the line and with an assigned length and angle.

Tool options:
Horizontal hor, horizontal

Draw a horizontal line from an assigned point defining the start, middle or end of the line and with an assigned length.

Tool options:
Vertical ver, vertical

Draw a vertical line from an assigned point defining the start, middle or end of the line and with an assigned length.

Tool options:
Rectangle rec, rect, rectangle

Draw a rectangle by assigning the points of two diagonally opposite corners.
Parallel through Point pp, ptp

Draw a given number of lines parallel to a selected existing line through an assigned point. Move your mouse closer or farther away from the point to create less or more distance between lines.

Tool options:
Parallel o, pa, offset, parallel

Draw a given number of lines parallel to a selected existing line with a given distance between lines.

Tool options:
Bisector bi, bisect

Draw a given number of lines bisecting two existing non-parallel lines (e.g. at an angle to each other with or without a common point).

Tool options:
Tangent (P,C) tanpc, tangentpc Draw a line from an assigned point tangent to an existing circle.
Tangent (C,C) none Draw a line tangent to two existing circles.
Tangent Orthogonal none Draw a line tangent to an existing circle and perpendicular to an existing line.
Orthogonal ortho, perp

Draw a line of a given length perpendicular to an existing line placing the center at an assigned point.

Tool options:
Relative Angle none

Draw a line with a given length and at a given angle relative to an existing line placing the center of the line at an assigned point.

Tool options:
Polygon (Cen,Cor) pl, polyline

Draw a polygon with a given number of sides assigning the center point and point of one vertex.

Tool options:
Polygon (Cen,Tan) none

Draw a polygon with a given number of sides assigning the center point and point of the center of one side.

Tool options:
Polygon (Cor,Cor) poly2, polygon2v

Draw a polygon with a given number of sides assigning the two points of one side.
